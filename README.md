# Introducing Repository
-This repository contained simple video recoder using OpenCV.

# How To Operaite
-Space bar = starting recoding.<br/>
-ESC = stop program.<br/>
-Mouse Leftbutton = zoom in.<br/>
${\textsf{\color{red}-Check the storage path when using it.}}$
